# YouTube UI Clone 🎬

A fully responsive YouTube home page interface built using modern HTML and CSS practices. This project replicates the layout and interactivity of the YouTube platform.

## 🚀 Live Demo
*Link will be available once GitHub Pages is enabled.*

## ✨ Features
* **Responsive Header**: Includes a search bar, voice search, and action icons.
* **Dynamic Sidebar**: A collapsible-style navigation menu.
* **Video Grid**: Responsive thumbnail layout using CSS Grid.
* **Interactive Tooltips**: Custom tooltips built with pure CSS.

## 🛠️ Built With
* **HTML5**: Semantic structure.
* **CSS3**: Advanced techniques including:
    * **CSS Grid**: For the main gallery.
    * **Flexbox**: For header alignment.
    * **Positioning**: Fixed headers and sidebars.

## 📂 Project Structure
* `channel-pictures/` — Profile images
* `icons/` — UI icons
* `styles/` — Organized CSS files
* `thumbnails/` — Video preview images
* `youtube.html` — Main entry point
