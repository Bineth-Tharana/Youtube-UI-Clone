#YouTube UI Clone 🎬

A fully responsive YouTube home page interface built using modern HTML and CSS practices. This project focuses on replicating the layout, styling, and interactivity of the YouTube web platform, showcasing advanced CSS techniques like Flexbox and CSS Grid.

✨ Features
Responsive Header: Includes a functional search bar, voice search button, and user action icons.

Dynamic Sidebar: A collapsible-style navigation menu for a clean user interface.

Video Grid: A fully responsive layout of video thumbnails that adjusts based on screen size.

Interactive Tooltips: Custom-built tooltips for header icons using CSS.

Modern Styling: Utilizes the Roboto font and precise icon sets to match the official YouTube aesthetic.

🛠️ Built With
HTML5: Semantic structure for better accessibility and SEO.

CSS3: Advanced layout techniques including:

CSS Grid: For the main video gallery.

Flexbox: For header and sidebar alignment.

Positioning: Fixed headers and sidebars for a seamless scrolling experience.
